# The Inner Sparkle

Marketing website for The Inner Sparkle's Voice AI, CoastaFlow CRM, and CoastaClaw digital employee services.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality Checks

```bash
npm run lint
npm run build
```

The production build is a static export in `out/`. It includes `index.html` and `robots.txt` and can be deployed directly to Vercel.
